# Dari Untuk Dengan Ucapan

A simple Google Apps Script project that automatically sends personalized emails from Google Form submissions.

When someone submits a Google Form, the response is stored in a Google Sheet. This script periodically checks for new responses, sends an HTML email to the recipient, and marks the row as processed to avoid sending duplicate emails.

## Workflow

```text
Google Form
      │
      ▼
Google Sheets
      │
      ▼
Google Apps Script (Time Trigger)
      │
      ▼
Send HTML Email
      │
      ▼
Mark row as "YES"
```

## Features

- Automatic email sending
- Google Forms integration
- Google Sheets integration
- HTML email template
- Random background image
- Supports multi-line messages
- Prevents duplicate emails
- Runs automatically using a time-based trigger

## Spreadsheet Format

The spreadsheet should contain the following columns.

| Column | Name | Description |
|-------:|------|-------------|
| A | Timestamp | Automatically generated by Google Forms |
| B | Dari | Sender name |
| C | Untuk | Recipient email address |
| D | Dengan Ucapan | Message |
| E | Template | Reserved for future templates |
| F | *(optional)* | Unused |
| G | Status | `YES` if the email has already been sent |

Example:

| Timestamp | Dari | Untuk | Dengan Ucapan | Template | Status |
|-----------|-------|--------|----------------|----------|--------|
| 6/30/2026 22:02:39 | Anonymous | ardhian777@gmail.com | kamu ganteng | Default | YES |

## How It Works

For every row:

1. Read the message from **Dengan Ucapan**.
2. Check whether **Status** is `YES`.
3. If not:
   - Generate an HTML email.
   - Select a random background image.
   - Preserve line breaks.
   - Send the email to **Untuk**.
4. Update **Status** to `YES`.

## Setup

### 1. Create a Google Form

Create a Google Form with the following fields:

- Dari
- Untuk
- Dengan Ucapan
- Template

The **Untuk** field should contain the recipient's email address.

---

### 2. Link the Form to Google Sheets

Open the **Responses** tab in Google Forms and create a linked spreadsheet.

---

### 3. Open Apps Script

From the spreadsheet:

```
Extensions
→ Apps Script
```

---

### 4. Replace the Default Code

Delete the default contents of `Code.gs` and paste the code from this repository.

---

### 5. Update the Spreadsheet URL

Replace:

```javascript
SpreadsheetApp.openByUrl("YOUR_SPREADSHEET_URL")
```

with your own spreadsheet URL.

---

### 6. Save the Project

Save the Apps Script project.

---

### 7. Grant Permissions

Run the function once manually.

Google will ask for permissions to:

- Read Google Sheets
- Send emails using Gmail

Approve all required permissions.

---

### 8. Create a Time Trigger

In Apps Script:

```
Triggers
→ Add Trigger
```

Choose:

- Function: `dailyNotification`
- Event Source: Time-driven
- Frequency: Every minute (or any interval you prefer)

The script will now run automatically.
